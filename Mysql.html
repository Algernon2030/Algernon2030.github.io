Mysql

2020-06-14 | SQL | |
|

主要内容为SQL语法知识

一：SQL的语法要求：

  • SQL语法可以多行也可以单行，主要以分号结尾。
  • 关键字不区分大小写

二：SQL的分类：

  • DDL：数据定义语言，定义数据库的对象：库、表、列
  • DCL：数据控制语言，定义数据库的记录
  • DQL：数据查询语言，执行查询操作

主要讲DQL语言：不改变数据，仅仅将数据展示到客户端。

语法：

SELECT #查询的列的名称

FROM #查询表的名称

WHERE #限定条件（关系和逻辑）

  • 关系条件：> < = !=
  • 逻辑条件：AND OR NOT BETWEEN ANF IFNULL

GROUP BY #对结果分类

ORDER BY #对数据进行排序

三：具体例子

  • 条件查询

案例1：查询工资>10000的员工信息

案例2：查询 job_id不是’IT_PROG’的员工姓名、job_id、salary

案例3：查询工资>10000并且小于20000的员工姓名、工资、年薪

案例4：查询部门编号不在90到120之间的并且 工资>15000的员工信息

  • 模糊查询

案例1：查询员工信息表中姓名第一个字符为e的员工信息

案例2：查询员工信息表中姓名第二个字符为e，第五个字符为a的员工信息

案例3：查询姓名中前面字符为ernst的员工信息

案例4：查询姓名中第二个字符为_的员工信息

  • 字段控制查询

使用DISTINCT、IFNULL

  • 给列名命名方式
  • 排序

单个字段、多个字段、表达式、别名、函数

其中默认为升序ASC，降序为DESC
