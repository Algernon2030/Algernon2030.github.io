新月专栏今天开通了，由前端的权威专家张新月为我们讲解前端的相关知识，张老师毕业于全国重点88所院校，最终留校读研，在前端领域可以说在他们专业中排第二，没人敢当第一。接下来的几个月，张老师会为我们详细讲解前端领域中的重点。

1 网页

1.1 网页是什么

由图片、声音、文字、视频等组成，HTML文件

1.2 HTML

超文本标记语言，专门制作网页，由标签组成

  • 超文本：
    可以加入图片、声音、动画等（超越文本限制）
    可以从一个文件跳转到另一个文件（超级链接文本）

1.3 网页的形成

前端人员开发代码 -> 浏览器显示代码（解析、渲染）-> 生成web页面
浏览器内核：读取网页内容，显示页面

2 Web标准

浏览器不同，显示页面有差异
主要包括结构、表现、行为三个方面

  • 结构：对网页元素整理分类，HTML

  • 表现：设置外观样式，CSS

  • 行为：交互的编写，JS

1 HTML语言规范

1.1 基本语法

HTML标签是由尖括号包围的关键词，例如<html>

标签经常成对出现，<html>为开始标签，</html>为结束标签

单标签，例<br />

1.2 标签关系

  • 包含关系
  • 并列关系

2 HTML基本结构标签

2.1 第一个页面

每个页面都有一个基本结构标签（也称为骨架标签）

在记事本中输入以上标签，将文件格式改为.html，就可打开一个页面

3 网页开发工具

VScode

ctrl+加号键/减号键：放大/减小视图

输入！按Tab键：生成见面骨架结构

右键“open in default browser”：预览页面

Auto Rename Tag扩展：同时修改开始、结束标签

3.1 文档类型声明标签

<!DOCTYPE> 告诉浏览器使用哪种HTML版本来显示网页

<!DOCTYPE html>表示当前页面采取HTML5版本显示网页

  • 位于文档最前面，在<html>标签之前
  • 不是HTML标签，只是文档类型声明标签

3.2 lang语言种类

lang="en"定义当前文档显示语言

  • en定义为英语
  • zh-CN定义为中文

3.3 字符集

在<head>标签内，通过<meta>标签的charset属性来规定HTML文档应使用哪种字符编码

charset常用的值有：GB2312、BIG5、GBK和UTF-8，常用UTF-8。

不写会乱码！

4 HTML常用标签

4.1 标签语义

4.2 ==标题标签==<h1>-<h6>

h为head的缩写

4.3 ==段落和换行标签==

<p>段落标签</p>

标签语义：把HTML文档分为若干段落

  • 一个段落会根据浏览器窗口大小自动换行

  • 段落之间有空隙

换行标签<br/>

break的缩写

标签语义：强制换行

  • 单标签
  • 开始新的一行，不产生空隙

4.4 文本格式化标签

斜体、粗体、删除线、下划线

标签语义：突出重要性

加粗：<strong></strong>或者 <b></b>
倾斜：<em></em> 或者<i></i>
删除线：<del></del> 或者<s></s>
下划线<ins></ins> 或者<u></u>

  • 都推荐用第一组标签

4.5 <div>和<span>标签

没有语义，只是一个盒子，用来装内容

<div>这是头部</div>

<span>今日价格</span>

  • <div>标签用来布局，一行只能放一个
  • <span>标签用来布局，一行可以放多个

4.6 ==图像标签和路径==

4.6.1 图像标签

<img>标签用于定义HTML页面中的图像

<img src="图像URL"/>

src是<img>标签的必须属性，用于指定图像文件的路径和文件名

图像标签的其他属性：

alt替换为文本，图像显示不出来的时候用文字替换

title提示文本，鼠标放到图片上，显示的文字

width图像的宽度

height图像的高度

border图像边框粗细

  • 图像标签可以写多个属性，必须在标签名后面
  • 属性之间不分顺序，以空格隔开
  • 键值对的格式，即属性=”属性值”
