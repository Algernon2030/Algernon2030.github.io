Java对象和类

2020-05-20 | Java编程 | |
|

Java作为面向对象的语言，主要支持下面的基本概念。

多态；继承；封装；抽象；类；对象；实例；方法；消息解析；

本章重点在于对象和类的概念。

对象：对象是类的实例，有状态和行为。例如一只猫作为对象，其中猫的状态为品种、颜色。猫的行为为摇尾巴、打哈欠。

类：类是一个模板，包含对象的行为和状态。

其中软件对象的状态就是属性，行为通过方法体现。

  • 方法可以操作对象内部状态的改变
  • 对象之间的相互调用通过方法完成

局部变量：在方法、构造方法定义的变量称为局部变量。变量声明和初始化都在方法中，方法结束后，变量就会自动销毁。

成员变量：成员变量定义在类中，在方法体外面，变量在创建对象的时候实例化，可以被风阀的语句块进行访问。

类变量：生命在类中，方法体之外，声明必须为static类型。

上文中代码里面的barking()，sleeping()，hungry()都是类中的方法。

构造方法：每个类都有构造方法，若没有显示的为类构造方法，编译器自动创建和类名一样的方法。一个类可以构造多个方法。

创建对象：

  • 声明：声明一个对象，包括对象的名称和类型
  • 实例化：使用关键字new来创建一个对象
  • 初始化：使用new创建对象，会调用构造方法初始化对象
